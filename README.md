# Family Connect

PT/OT/ST family communication tool for Directors of Rehabilitation in assisted living. Generate professional emails across every stage of the therapy lifecycle.

Live: https://cliff-peters.github.io/family-connect

## Tabs

**Therapy Start** — New referral emails for existing residents, new move-ins, and consent follow-ups. Supports PT, OT, and ST with discipline-specific language and consent nudge options.

**Post-Eval** — Post-evaluation emails with per-discipline goal selection, ST swallow/cognition focus, and visit frequency. Adapts for single and multi-discipline plans.

**Therapy Update** — Progress update emails with status indicators, custom comments, and frequency change options per discipline.

**Therapy Transition** — Transition emails for goals met, visit preservation, plateau, and home health. Includes Part A/Part B billing language.

**Fall Follow-Up** — Post-fall emails with intervention details, equipment links, and purchasing options.

**Quick Reference** — Talking points for family calls covering Health & Wellness, insurance visit preservation, and common FAQs.

## Features

- Facility name field populates every email automatically
- No data stored or transmitted — runs entirely in the browser
- Single HTML file, no dependencies
